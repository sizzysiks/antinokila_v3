    Image distortion

    This scene consists of image textures. When the user scrolls the page the images are gradually distorted into an animated abstact shape of their original form, creating a pleasing progress effect when navigating the page.
    Three.js , GLSL Shaders , Post-Proccesing

    Customizable object

    This is an interactive scene of an abstract object.The object consists of three different layers. The user can modify the shape of each of them along with the animation speed using the control panel.
    Three.js , GLSL Shaders , dat.GUI

    Mouse displacement

    Highjacking the mouse to alter the shape & color of the existing texture when the mouse hovers over it. This creates a pleasing wave like distortion of the original image
    Three.js , GLSL Shaders

    Physics based object movement

    An object generetor which creates new objects on user click. Each object falls from the 'sky' and interacts with the the other objects as a real life object would, hitting other objects and bouncing accordingly.
    Three.js , Cannon.js

    3D Model

    Rendering a glTF 3D model of a shoe, which the user can view from all different angles. This can be used for promoting a client's product in a unique and interactive way.
    Three.js , Blender , glTF
